Promotion

Al kommunikation til markedet — reklame, PR, sociale medier

Promotion er al den kommunikation virksomheden laver for at påvirke kunderne. Det er det mest synlige P i de 4 P'er.

📐 Vigtigste modeller

Promotionmix

Hvad? De forskellige kommunikationsværktøjer virksomheden kan bruge

Hvorfor? Forskellige værktøjer når forskellige målgrupper

Hvordan?
• Reklame = betalt massekommunikation (TV, online, print)
• PR (Public Relations) = gratis omtale, pressemeddelelser
• Sales Promotion = tilbud, konkurrencer, rabatkuponer
• Personligt salg = direkte sælger-kunde kontakt
• Direct marketing = målrettet kommunikation (email, SMS)
• Online promotion = sociale medier, SEO, content marketing

AIDA-modellen

Hvad? 4 faser i kommunikationens virkning: Attention → Interest → Desire → Action

Hvorfor? Bruges til at vurdere om reklamen virker i alle faser

Hvordan? A = fang opmærksomhed. I = skab interesse. D = skab lyst/behov. A = få til handling (køb!).

Sociale medier & Content Marketing

Hvad? Markedsføring via sociale platforme og værdiskabende indhold

Hvorfor? Her er kunderne! Organic reach + betalt annoncering

Hvordan? Content = blogs, videoer, guides der skaber værdi. Sociale medier = dialog og deling.

🎯 Eksamentips

  • ⚡Kend forskellen: Reklame (betalt) vs. PR (gratis omtale)
  • ⚡AIDA: Attention → Interest → Desire → Action
  • ⚡Paid media = betalt. Owned media = ejet (hjemmeside). Earned media = fortjent (omtale/deling)
  • ⚡SEO = søgemaskineoptimering. SEM = betalt søgeannoncering
  • ⚡Influencer marketing = betaler personer med følgere for at promovere

💡 HUSK: Promotion = Reklame + PR + Sales Promotion + Personligt salg + Digital (AIDA + Paid/Owned/Earned)
