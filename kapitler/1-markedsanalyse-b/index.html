Markedsanalyse

Hvordan man indsamler og analyserer data om markedet

Markedsanalyse handler om at indsamle information om kunder, konkurrenter og marked — så virksomheden kan træffe bedre beslutninger.

📐 Vigtigste modeller

Kvalitativ vs. Kvantitativ

Hvad? To grundlæggende metoder til dataindsamling

Hvorfor? Forskellige spørgsmål kræver forskellige metoder

Hvordan? Kvalitativ = dybde, interview, fokusgrupper (HVORFOR). Kvantitativ = tal, spørgeskemaer, statistik (HVOR MANGE).

Primær vs. Sekundær data

Hvad? Har man selv indsamlet data (primær) eller bruger man eksisterende (sekundær)?

Hvorfor? Sekundær = billigt og hurtigt. Primær = præcist til formålet.

Hvordan? Primær = egen undersøgelse. Sekundær = Danmarks Statistik, rapporter, databaser.

Repræsentativitet

Hvad? Om stikprøven ligner hele målgruppen (populationen)

Hvorfor? Hvis den ikke er repræsentativ, kan resultaterne ikke bruges!

Hvordan? Stikprøven skal være: stor nok + tilfældigt udvalgt + dække alle undergrupper.

🎯 Eksamentips

  • ⚡Kvalitativ = dybde, FÅ respondenter, interview/fokusgrupper
  • ⚡Kvantitativ = bredde, MANGE respondenter, spørgeskema
  • ⚡Primær data = indsamlet selv. Sekundær = allerede eksisterende
  • ⚡Stikprøve skal være repræsentativ for at resultaterne er gyldige

💡 HUSK: Markedsanalyse = Kvalitativ/Kvantitativ + Primær/Sekundær + Repræsentativitet
